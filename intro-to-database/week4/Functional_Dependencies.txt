[Q1] Consider relation R(A,B,C,D,E) with functional dependencies:

AB → C, C → D, BD → E

Which of the following sets of attributes does not functionally determine E?

1 BE 
2 ABC 
3 AD
4 AB 

[Q2] Consider relation R(A,B,C,D,E) with functional dependencies:

D → C, CE → A, D → A, AE → D

Which of the following is a key?

1 A 
2 BDE 
3 BD
4 CDE 

[Q3] Let relation R(A,B,C,D,E,F,G,H) satisfy the following functional dependencies:

A → B, CH → A, B → E, BD → C, EG → H, DE → F

Which of the following FDs is also guaranteed to be satisfied by R?

1 CDE → AF 
2 CGH → BF 
3 ADG → CH 
4 BCD → FH

3 ADEF 
4 ABE

[Q5] Consider relation R(A,B,C,D,E,F,G) with functional dependencies:

[Q6] Let relation R(A,B,C,D,E) satisfy the following functional dependencies:

[Q7] Let relation R(A,B,C,D) satisfy the following functional dependencies:

A → B, B → C, C → A

3 B → A, B → C, C → B 
4 A → B, B → A, C → A
